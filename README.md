# ??? Vecinity - Marketplace Pilar

> **"Soltá lo que no usás, encontrá lo que necesitás"**

**Vecinity** es una plataforma comunitaria diseñada específicamente para barrios cerrados y comunidades privadas en Pilar, Argentina. El objetivo es facilitar el intercambio, la venta de productos y la conexión de servicios entre vecinos de forma segura y directa.

---

## ?? Características Principales

* **?? Marketplace Global y Local:** Publicá productos para vender, canjear (trueque) o regalar. Filtrá por tu barrio o buscá en todo Pilar.
* **??? Directorio de Servicios:** Encontrá profesionales recomendados por tus propios vecinos (electricistas, plomeros, médicos, etc.).
* **?? Mascotas y Comunidad:** Espacios dedicados para reportar mascotas perdidas, compartir novedades del barrio y hasta intercambiar recetas.
* **? Sistema de Reputación:** Calificá y dejá reseñas a otros usuarios para construir una comunidad confiable.
* **?? Contacto Directo:** Integración con WhatsApp para concretar transacciones con un solo clic.
* **??? Panel de Administración:** Control total sobre usuarios y publicaciones para mantener la armonía del sitio.

---

## ??? Tecnologías Utilizadas

Este proyecto es una **Single Page Application (SPA)** construida con:

* **HTML5/CSS3** (Estructura y estilos personalizados).
* **Tailwind CSS 3.4** (Diseño responsivo y moderno).
* **Lucide Icons** (Iconografía minimalista).
* **JavaScript (Vanilla)** (Lógica de autenticación, navegación y filtrado).
* **Google Fonts** (*Playfair Display* para elegancia y *DM Sans* para legibilidad).
